I need pride becuase...

An extensive tutorial to help walk you through my process of creating the photos.

Some steps may be unnessacray, but I've included everything so make this as esay as possible. The best way to navigate this is to make your browser as narrow as possible.

Before we begin, download the following files into a single folder on your desktop.

  1. ineedpridebecuase.zip
  2. 10BLACK.png
Double click the ineedpridebecuase.zip. This will unpack the photos and allow you to open them.

Begin with opening the sample photo. To open the photo, go to File - Open: and select the folder that you saved on your desktop. Double click on the photo you want ot edit. Take note of the lighting. This photo is taken on a dull backgrounds, which means we have to brighten the photo and make sure we stay true to the values of color that are present.

To begin this process, check the levels of the photo. The levels pallet can be found by going into Image - Ajustments - Levels.

I've adjusted the levels to just touching each edge of the histogram. This is as far as they should go, in order to keep the skin tones and colours natural.

You should have something like this:

Now we need to crop the photo to give it a better format. The best way to do this is to look at how the photo is set up. Some of the photos have the board set to the left, or have two people in them. The main focus for these images is the face and the board. All other details are minor.

Use the crop tool and erase the values in the width and height area.

As you can see, the photo is not square. The height is much more then the width, but this will be evened out a touch when we add the boarder.

But we need to make this photo black and white first.

Select the black and white tool. Image - Adjustments - Black and White

The photo is dull and very flat. In order to adjust this, we need to look at the values in the image. Zac is wearing a pink shirt, which is the dullest value in this image. we can darken that to give the board and his skin some contrast.

There are two ways to navigate values. You'll notice that when you turn the photo black and white, there are values automatically imputed. These are good values, but in some cases, like with Zac's photo, we need to tweak it.

(I have the newest version of photoshop, if it dosen't look indentical it's alright.) The lowest value is magentas.

Zac's hair and skin both have a red undertone. So by adjusting the red value, we can change his main colour.

This is the result of making the red darker.

This image is way to dark, and takes away the details in his shirt and hair. Instead of adjusting a value that encompases so much of the picture, it's much easier to tweak that value to something a bit lighter. Once the overall value is okay, you can change a more specific colour.

There won't be a huge difference in the image, but it's enough to keep the values true to the colour. Now look for the part you want to make lighter or darker. Once you've choosen, notice the most prominate colour from your pallete. If you're not sure, turn of the black and white layer, and play with the Hue and Saturation Pallete. Image - Adjustments - Hue/Saturation

Now that you know which colour to work with, adjust it to make the contrast that looks natural and compliments the lights or darks of the photo.

The result is a crisper photo and better contrast. The one on the right has the magenta adjustment.

Now that the photo is how we'd like it, we can create the border.

First, we need to merge the layers together to create one single layer. The best way to do this make a copy of the background layer, and then select the black and white layer/levels layer/image.

Right click and select Merge Layers

Rename this layer to Maing Image. Copy the background layer again, and trash the layer with a lock on it. Now create a new layer in between the black and white image and the colour image.

Now select the blank layer and go to Image - Canvas Size

The canvas size will depend on the your final crop. I made this one 6.5" x 9". Make the height at least 3 inches bigger then the image size, and the width about half an inch on each side.

In the new layer you have created, make a white rectangle that covers the entire canvas.

If the rectangle isn't white, double click on the rectangle. A colour window with colours will come up.

White is #ffffff in this area.

Move the image to the middle and to the top of the background.

Now you can delete the colour layer. At this point we need to add the branding and logo to the image. File - Place - 10BLACK.png This image should place in the middle, and by holding shift you can resize it to about a quarter of the image size. Hit enter. click the move tool.

Move the logo to the right side of the image.

Now that we have the completed image, click File - Save as - and create a new folder to save the images in called Final Images. Change the file format to PNG and click save.
