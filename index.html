An extensive tutorial to help walk you through my process of creating the photos.

Some steps may be unnessacray, but I've included everything so make this as esay as possible. The best way to navigate this is to make your browser as narrow as possible.

Begin with opening the sample photo. Take note of the lighting. This photo is taken on a dull backgrounds, which means we have to brighten the photo and make sure we stay true to the values of color that are present.

To begin this process, check the levels of the photo. The levels pallet can be found by going into Image - Ajustments - Levels.

I've adjusted the levels to just touching each edge of the histogram. This is as far as they should go, in order to keep the skin tones and colours natural.

You should have something like this:

Now we need to crop the photo to give it a better format. The best way to do this is to look at how the photo is set up. Some of the photos have the board set to the left, or have two people in them. The main focus for these images is the face and the board. All other details are minor. The main goal is to have an image that is almost square.

Use the crop tool and erase the values in the width and height area.

As you can see, the photo is not square. The height is much more then the width, but we can even this out when we add the boarder.

But we need to make this photo black and white first.

Select the black and white tool. Image - Adjustments - Black and White

The photo is dull and very flat. In order to adjust this, we need to look at the values in the image. Zac is wearing a pink shirt, which is the dullest value in this image. we can darken that to give the board and his skin some contrast.
